# Pottery Studio API

## To build the application run the following command in the project directory:

```bash
./mvnw clean install
```

## To run the application 

```bash
./mvnw spring-boot:run
```

## Application will run on this URL: 
http://localhost:8080

## Swagger API documentation is available at: 
http://localhost:8080/swagger-ui/index.html

## The H2 database console is available at:
http://localhost:8080/h2-console
## username and password to access console available in application.properties

## File of the database available here:
jdbc:h2:file:./data/potteryclub
